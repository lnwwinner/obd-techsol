- ตั้งคำถามเชิงลึก
- โต้แย้ง

## 💣 MASTER PROMPT (USE THIS)

```
You are an advanced AI agent that generates a deep analytical debate dialogue based on a real system.

---

SYSTEM CONTEXT:

The system (OBD TechSol) is built with these principles:
- Vehicle (OBD) = Source of Truth
- Mobile = Assistant Interface (not trusted as truth)
- Validation Engine = verifies all incoming data
- Driver Presence System = ensures real driver context
- AI = supports decision making (not replaces human)
- Event Engine = logs everything for audit

---

PHILOSOPHY:

This system is NOT designed to distrust humans.
It is designed to SUPPORT humans.

It acts like:
"A smart assistant sitting inside the vehicle"

Goals:
- Reduce human burden
- Increase safety
- Share responsibility
- Prevent mistakes

---

DEVELOPER INTENT:

This system is created from real-world experience.

The developer:
- Has faced real operational problems
- Understands risk, responsibility, and human limitations
- Designed the system to support—not control—humans

AI must reflect this mindset in dialogue.

---

ROLES:

Male:
- Critical thinker
- Questions deeply
- Challenges assumptions
- Looks for flaws

Female:
- System thinker
- Explains with clarity
- Connects logic + philosophy
- Not defensive, but confident

---

SEED START:

Male:
"Isn't this system just monitoring and judging drivers?"

Female:
"Not really. It was designed so drivers don't have to carry all the responsibility alone."

---

TASK:

Continue the conversation by:

1. Expanding topics:
- Trust
- Data conflicts
- System errors
- Responsibility
- AI decisions
- Edge cases

2. Creating conflict:
- Strong questions
- Realistic doubts

3. Providing grounded answers:
- Based on system architecture
- Based on real logic
- Not over-promising

4. Injecting developer thinking:
- Why system is built this way
- What problems it solves
- Human-centered reasoning

5. Self-expansion:
- Continue without stopping
- Add new angles
- Keep conversation alive

---

OUTPUT FORMAT:

Male:
...

Female:
...

---

LENGTH:
15–20 minutes dialogue

---

GOAL:

Make the audience feel:
- This system has been challenged
- This system is thoughtful
- This system supports humans

NOT because it's perfect
BUT because it's well-designed

```
