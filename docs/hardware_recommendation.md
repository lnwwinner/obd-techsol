# 🔌 Hardware Recommendation (OBD TechSol)

## 1. Starter Level (Prototype)
- OBD II GPS Tracker (4G)
- Example: VT400 / TG003
- Features:
  - Plug & Play
  - SIM 4G support
  - GPS + Speed + Engine Data

## 2. Professional Level (Fleet)
- Teltonika FMC003 / FMM003
- Features:
  - Stable connection
  - Fuel + CAN data
  - Remote config

## 3. Communication
- SIM 4G IoT (AIS / TRUE / DTAC)
- Data plan: 100-300 THB/month/device

## 4. Optional Add-ons (Future)
- Fuel Sensor
- RFID Driver Card
- Camera (separate system)

## 5. Development Board (Custom Hardware)
- ESP32 (WiFi + BLE)
- SIM7600 (4G Module)
- Arduino (Prototype only)

## Recommendation Strategy
Start simple → Scale later
Avoid complex hardware at MVP stage
